# GSD Framework Skill

Reference: `starter-kits/gsd-framework/SKILL.md`

## Quick Commands

| Command | Purpose |
|---------|---------|
| `/gsd:progress` | Check current status |
| `/gsd:plan-phase N` | Plan tasks for phase N |
| `/gsd:execute-plan` | Execute current plan |
| `/gsd:research-phase N` | Research unknowns before planning |
| `/gsd:verify-work` | Guide manual user acceptance testing |

## Key Files

| File | Purpose |
|------|---------|
| `.gsd/PROJECT.md` | Project vision and core definition |
| `.gsd/ROADMAP.md` | Phase breakdown |
| `.gsd/PLAN.md` | Current phase tasks |
| `.gsd/STATE.md` | What exists now |
| `.gsd/SUMMARY.md` | Daily progress |
| `.gsd/ISSUES.md` | Blockers and deferred items |

## Workflow

1. **Start**: `/gsd:progress` to check where you are
2. **Plan**: `/gsd:plan-phase N` to plan next phase
3. **Execute**: `/gsd:execute-plan` to run the plan
4. **Verify**: `/gsd:verify-work` for UAT
